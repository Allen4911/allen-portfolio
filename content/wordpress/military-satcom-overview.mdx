## Overview

Military Satellite Communication (MILSATCOM), unlike commercial satellite communications, has **Anti-Jamming (AJ)**, **Low Probability of Intercept/Detection (LPI/LPD)**, and **high availability** as core requirements.

## Frequency Band Characteristics

| Band | Frequency | Characteristics | Primary Use |
|------|-----------|-----------------|-------------|
| UHF | 300~3000 MHz | Excellent mobility, small antenna | Tactical terminals, mobile platforms |
| SHF (X) | 7.25~8.4 GHz | Balanced performance | Basic tactical communications |
| SHF (Ka) | 26.5~40 GHz | High capacity, rain susceptibility | Strategic comms, broadband |
| EHF | 30~300 GHz | Best anti-jamming | Core strategic communications |

## Link Budget Fundamentals

Satellite communication link budget calculates losses and gains as a signal travels ground вҶ’ satellite вҶ’ ground.

```
EIRP(dBW) - FSL(dB) - OtherLoss(dB) + G/T(dB/K) - k(dBW/K/Hz) = C/NвӮҖ(dBHz)

Where:
  EIRP = Equivalent Isotropically Radiated Power (Tx Power + Antenna Gain)
  FSL  = Free Space Loss = 20В·log(4ПҖd/О»)
  G/T  = Receive figure of merit (Antenna Gain - Noise Temperature)
  k    = Boltzmann constant (-228.6 dBW/K/Hz)
```

### Example Calculation (GEO Satellite, X-band)

```python
import math

def free_space_loss_dB(distance_km, freq_GHz):
    """Calculate free space path loss"""
    c = 3e8  # m/s
    f = freq_GHz * 1e9
    d = distance_km * 1e3
    lam = c / f
    fsl = 20 * math.log10(4 * math.pi * d / lam)
    return fsl

# GEO satellite (~36,000 km), X-band 8 GHz
fsl = free_space_loss_dB(36000, 8.0)
print(f"Free Space Path Loss: {fsl:.1f} dB")  # вүҲ 205.6 dB
```

## Anti-Jamming Techniques

### Frequency Hopping

```
Time вҶ’
Channel: f3 | f7 | f1 | f9 | f4 | f2 | ...
            вҶ‘                           вҶ‘
    Hopping pattern determined by pseudo-random sequence (shared key required)
```

### Spread Spectrum

- **FHSS**: Frequency Hopping вҖ” rapid movement across narrow bands
- **DSSS**: Direct Sequence вҖ” band spreading with PN code, resistant to jamming
- **Chirp SS**: Linear frequency modulation вҖ” low probability of detection

|--------|----------|------|-------------|
| AEHF | USA | EHF | Nuclear survivability, top-secret comms |
| WGS | USA | X/Ka | High-capacity broadband |
| SKYNET | UK | UHF/X/Ka | NATO interoperability |
| MUOS | USA | UHF | Mobile, 3G-based waveform |
| ANASIS-II | South Korea | X | Dedicated military satellite |

## Summary

Military satellite communication is centered on security, survivability, and anti-jamming capability compared to commercial systems. When developing modem SW, standard waveforms (MIL-STD-188 series) and encryption requirements (Type 1 cryptography) must always be considered.

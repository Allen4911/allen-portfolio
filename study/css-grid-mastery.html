CSS Grid — Complete Guide

Grid vs Flexbox decision tree, grid-template-areas for layout composition, auto-fit vs auto-fill, and responsive grid patterns.

CSS Grid — Complete Guide

Grid vs Flexbox — The Decision

One axis (row OR column)?      → Flexbox
Two axes (row AND column)?     → Grid
Unknown number of items, auto-wrap? → Grid with auto-fit
Centered single item?          → Either (grid: place-items: center)

They're complementary, not competing. Use grid for layout, flexbox for component internals.

The Two Axes

grid-template-columns defines the column axis (horizontal)
grid-template-rows    defines the row axis (vertical)

Columns →
         col 1    col 2    col 3
row 1  | cell | | cell | | cell |   ← Rows ↓
row 2  | cell | | cell | | cell |
row 3  | cell | | cell | | cell |

Defining the Grid

Explicit Columns and Rows

.container {
  display: grid;
  grid-template-columns: 200px 1fr 1fr;  /* Fixed | flexible | flexible */
  grid-template-rows: 100px auto;         /* Fixed height | content height */
  gap: 16px;                              /* gap between all cells */
  /* column-gap: 24px; row-gap: 8px;  — different values */
}

The fr Unit — Fractional Remaining Space

fr allocates the remaining space after fixed values are subtracted:

/* 3 equal columns */
grid-template-columns: 1fr 1fr 1fr;
/* Same as: */
grid-template-columns: repeat(3, 1fr);

/* Sidebar + content: sidebar fixed, content gets all remaining */
grid-template-columns: 280px 1fr;

/* 1:2:1 ratio */
grid-template-columns: 1fr 2fr 1fr;

The repeat() Function

/* 12 equal columns (common CSS grid system) */
grid-template-columns: repeat(12, 1fr);

/* 4 columns, each 200px */
grid-template-columns: repeat(4, 200px);

/* Pattern: 40px 200px 40px 200px ... (6 times = 12 tracks) */
grid-template-columns: repeat(6, 40px 200px);

auto-fit vs auto-fill — Responsive Without Media Queries

Both with minmax() enable responsive grids without a single media query:

/* auto-fit: collapses empty tracks, items stretch to fill */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

/* auto-fill: preserves empty track space */
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

Visual difference:

Container width: 900px, min item width: 250px → 3 items fit

With 2 actual items:

auto-fit:   [  item 1  ][  item 2  ]  ← stretches to fill
auto-fill:  [  item 1  ][  item 2  ][        ]  ← preserves ghost track

Rule of thumb: Use auto-fit (items stretch to fill gaps) for almost everything. Use auto-fill when you need empty cells to take physical space (e.g., calendar grids).

Named Template Areas — Layout as ASCII Art

The most readable way to define complex layouts:

.page {
  display: grid;
  grid-template-areas:
    "header header header"
    "nav    main   aside"
    "nav    main   aside"
    "footer footer footer";
  grid-template-columns: 200px 1fr 300px;
  grid-template-rows: auto 1fr 1fr auto;
  min-height: 100vh;
  gap: 0;
}

header { grid-area: header; }
nav    { grid-area: nav; }
main   { grid-area: main; }
aside  { grid-area: aside; }
footer { grid-area: footer; }

Use . for empty cells:

grid-template-areas:
  "header header ."
  "nav    main   .";

Placing Items Explicitly

Items auto-place by default. Override with line numbers or span:

/* Lines are numbered 1 to (n+1) — or -1 from the end */
.hero {
  grid-column: 1 / 4;        /* Span columns 1 to 4 (= 3 columns) */
  grid-column: 1 / -1;       /* From first to last line (full width) */
  grid-column: span 3;       /* Span 3 columns from current position */
  grid-row: 1 / 3;           /* Rows 1 to 3 */
}

/* Named lines */
.sidebar {
  grid-column: sidebar-start / sidebar-end;
}

Alignment — Aligning Items and Tracks

Grid has two sets of alignment properties:

/* Align all items within their cells */
align-items:   start | end | center | stretch;   /* vertical */
justify-items: start | end | center | stretch;   /* horizontal */
place-items:   center;  /* shorthand: vertical horizontal */

/* Align the grid tracks within the container */
align-content:   start | end | center | between | around | evenly;
justify-content: start | end | center | between | around | evenly;

/* Align a single item */
.item {
  align-self:   center;
  justify-self: end;
}

Centering a single element:

.container {
  display: grid;
  place-items: center;  /* Both vertically and horizontally */
}

Implicit Grid — Handling Extra Items

When items overflow the explicit grid, the implicit grid takes over:

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* Explicit: 3 columns */
  /* Implicit rows are auto-sized by default */
  grid-auto-rows: 200px;   /* Set height of implicit rows */
  grid-auto-flow: row;     /* dense — fills gaps aggressively */
}

Subgrid — Aligning Nested Elements

Subgrid lets a child element participate in its parent's grid tracks. Browser support is now solid (all major browsers 2024+).

/* Parent grid */
.card-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}

/* Card spans 1 column but uses 3 row tracks for internal alignment */
.card {
  display: grid;
  grid-row: span 3;               /* Take 3 parent rows */
  grid-template-rows: subgrid;    /* Inherit parent's row tracks */
}

/* Now all cards' headers/bodies/footers align across cards */
.card-header { /* Occupies row 1 of the subgrid */ }
.card-body   { /* Occupies row 2 — always aligned with other cards */ }
.card-footer { /* Occupies row 3 — always at the same height */ }

Common Layout Patterns

Holy Grail Layout (Header + Sidebar + Content + Sidebar + Footer)

.layout {
  display: grid;
  grid-template:
    "header" auto
    "nav main aside" 1fr
    "footer" auto
    / 200px 1fr 200px;
  min-height: 100vh;
}

Masonry-style (approximated)

.masonry {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  grid-auto-rows: 10px;  /* Small row height for dense packing */
}

/* JavaScript sets grid-row-end based on content height */
.masonry-item {
  /* grid-row-end: span N — calculated from clientHeight */
}

Responsive Card Grid (no media queries)

.cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(min(100%, 320px), 1fr));
  gap: 24px;
}
/* min(100%, 320px): items never exceed 320px but can be smaller on narrow screens */

Grid Quick Reference

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  grid-template-areas: "...";
  gap: 16px;                    /* row-gap column-gap */
  place-items: center;          /* align-items justify-items */
  place-content: center;        /* align-content justify-content */
  grid-auto-flow: row dense;
  grid-auto-rows: minmax(100px, auto);
}

.item {
  grid-column: 1 / 3;           /* or span 2 */
  grid-row: 1 / -1;             /* or span 3 */
  grid-area: header;            /* named area */
  place-self: center end;       /* align-self justify-self */
}

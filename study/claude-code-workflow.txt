4:T833,# AI-Assisted Development with Claude Code

## Overview

Claude Code is Anthropic's official CLI that lets you run Claude directly in your terminal. Combined with tmux and a Telegram bridge, you can build a fully remote-controlled AI development team — coordinating multiple Claude instances as specialized agents from your phone.

## Installing Claude Code

```bash
npm install -g @anthropic/claude-code
claude --version
```

Authenticate with your Anthropic API key:

```bash
export ANTHROPIC_API_KEY=sk-ant-...
claude auth login
```

## Setting Up a tmux Team

The core idea: run multiple Claude Code sessions in separate tmux panes, each assigned a specific role.

```bash
# Create a new session with 6 panes
tmux new-session -s team1
# Split into panes and assign roles:
# Pane 0: Team Lead (orchestrator)
# Pane 1: PM / Architect
# Pane 2: Researcher
# Pane 3: Designer
# Pane 4: Developer
# Pane 5: Reviewer
```

Each pane runs its own `claude` instance with a dedicated CLAUDE.md defining its role and responsibilities.

## Telegram Bridge Configuration

The OpenClaw Claude Bridge connects Telegram messages to specific tmux panes:

```bash
# Install the bridge
git clone https://github.com/bettep-dev/openclaw-claude-bridge
cd openclaw-claude-bridge
bash setup-team.sh
```

Message routing uses prefixes:
- `@cc 메시지` → Pane 0 (Team Lead)
- `@cc @민준 메시지` → Pane 1 (PM)
- `@cc @all 메시지` → broadcast to all panes

## RTK Token Optimization

RTK (Rust Token Killer) is a CLI proxy that filters verbose command output before it reaches Claude, reducing token usage by 60–90%.

```bash
# Install RTK
cargo install rtk
# RTK hooks intercept all shell commands automatically
rtk gain  # Show savings analytics
```

## Practical Workflow

1. Send task via Telegram: `@cc @민준 새 기능 설계해줘`
2. PM (Pane 1) designs the approach and routes sub-tasks
3. Developer (Pane 4) implements; Reviewer (Pane 5) audits
4. Results reported back via Telegram

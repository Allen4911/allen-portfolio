## 개요

군용 위성통신(Military Satellite Communication, MILSATCOM)은 상용 위성통신과 달리 **항재밍(Anti-Jamming)**, **저피탐(LPI/LPD)**, **고가용성** 요구사항이 핵심입니다.

## 주파수 대역별 특성

| 대역 | 주파수 | 특성 | 주요 용도 |
|------|--------|------|-----------|
| UHF | 300~3000 MHz | 기동성 우수, 소형 안테나 | 전술 단말, 이동체 |
| SHF (X) | 7.25~8.4 GHz | 균형적 성능 | 기본 전술 통신 |
| SHF (Ka) | 26.5~40 GHz | 고용량, 강우 영향 | 전략 통신, 브로드밴드 |
| EHF | 30~300 GHz | 항재밍 최강 | 핵심 전략 통신 |

## 링크 버짓 기본 구조

위성통신 링크 버짓은 신호가 지상 → 위성 → 지상을 거치면서의 손실/이득을 계산합니다.

```
EIRP(dBW) - FSL(dB) - 기타손실(dB) + G/T(dB/K) - k(dBW/K/Hz) = C/N₀(dBHz)

여기서:
  EIRP = 등가등방복사전력 (Tx Power + Antenna Gain)
  FSL  = 자유공간 경로손실 = 20·log(4πd/λ)
  G/T  = 수신 성능 지수 (안테나 이득 - 잡음 온도)
  k    = 볼츠만 상수 (-228.6 dBW/K/Hz)
```

### 예시 계산 (GEO 위성, X-band)

```python
import math

def free_space_loss_dB(distance_km, freq_GHz):
    """자유공간 경로손실 계산"""
    c = 3e8  # m/s
    f = freq_GHz * 1e9
    d = distance_km * 1e3
    lam = c / f
    fsl = 20 * math.log10(4 * math.pi * d / lam)
    return fsl

# GEO 위성 (약 36,000 km), X-band 8 GHz
fsl = free_space_loss_dB(36000, 8.0)
print(f"자유공간 경로손실: {fsl:.1f} dB")  # ≈ 205.6 dB
```

## 항재밍 기술

### 주파수 도약 (Frequency Hopping)

```
시간 →
채널: f3 | f7 | f1 | f9 | f4 | f2 | ...
       ↑                           ↑
    호핑 패턴: 의사 난수 시퀀스로 결정 (공유 키 필요)
```

### 확산 스펙트럼 (Spread Spectrum)

- **FHSS**: 주파수 도약 — 좁은 대역을 빠르게 이동
- **DSSS**: 직접 수열 — PN 코드로 대역 확산, 재밍에 강함
- **Chirp SS**: 선형 주파수 변조 — 저확률 탐지

мЈјмҡ” кё°мҲ  кіјм ң

1. лҢҖк·ңлӘЁ м „нҢҢ м§Җм—° (Large Propagation Delay)

LEO кё°мӨҖ нҺёлҸ„ мөңлҢҖ ~10 ms, GEOлҠ” ~270 msлЎң кё°мЎҙ м§ҖмғҒл§қ лҢҖл№„ л§Өмҡ° нҒ° RTTлҘј к°Җм§‘лӢҲлӢӨ.

мҳҒн–Ҙ л°Ҹ лҢҖмқ‘ л°©м•Ҳ:

л¬ём ң: HARQ Stop-and-Wait вҶ’ мІҳлҰ¬лҹү кёүлқҪ
  в””в”Җ LEO(600km) кё°мӨҖ RTT вүҲ 4 ms Г— 2 = ~8 ms
  в””в”Җ GEO кё°мӨҖ RTT вүҲ 270 ms Г— 2 = ~540 ms

лҢҖмқ‘:
  в‘  HARQ л№„нҷңм„ұнҷ” (Rel-17 NTN: HARQ мһ¬м „мҶЎ нҡҹмҲҳ нҷ•мһҘ)
  в‘Ў Extended HARQ Round-Trip Time н—Ҳмҡ©
  в‘ў ARQ мғҒмң„ кі„мёө(RLC AM)м—җм„ң мһ¬м „мҶЎ лӢҙлӢ№

2. лҸ„н”Ңлҹ¬ нҺёмқҙ (Doppler Shift)

LEO мң„м„ұмқҖ мҙҲмҶҚ м•Ҫ 7.5 km/sлЎң мқҙлҸҷн•ҳл©° мөңлҢҖ мҲҳмӢӯ kHzмқҳ мЈјнҢҢмҲҳ нҺёмқҙк°Җ л°ңмғқн•©лӢҲлӢӨ.

# лҸ„н”Ңлҹ¬ нҺёмқҙ кі„мӮ° мҳҲмӢң
def doppler_shift(freq_hz, velocity_ms, angle_deg):
    """
    freq_hz    : л°ҳмҶЎнҢҢ мЈјнҢҢмҲҳ (Hz)
    velocity_ms: мң„м„ұ мҶҚлҸ„ (m/s)
    angle_deg  : мң„м„ұ мқҙлҸҷ л°©н–Ҙкіј мӢңм„  л°©н–Ҙ к°ҒлҸ„
    """
    import math
    c = 3e8  # кҙ‘мҶҚ (m/s)
    theta = math.radians(angle_deg)
    delta_f = freq_hz * (velocity_ms / c) * math.cos(theta)
    return delta_f

# LEO мң„м„ұ (7500 m/s), Ka лҢҖм—ӯ (20 GHz), м •л©ҙ нҶөкіј мӢң
shift = doppler_shift(20e9, 7500, 0)
print(f"мөңлҢҖ лҸ„н”Ңлҹ¬ нҺёмқҙ: {shift/1e3:.1f} kHz")  # вүҲ 500 kHz

# к°ңл…җм Ғ AI мұ„л„җ мҳҲмёЎ нқҗлҰ„
class NTNChannelPredictor:
    """
    мң„м„ұ к¶ӨлҸ„ м •ліҙ + мёЎм • н”јл“ңл°ұ вҶ’ лӢӨмқҢ мҠ¬лЎҜ мұ„л„җ мҳҲмёЎ
    """
    def predict(self, ephemeris, prev_csi, weather_info):
        # 1. к¶ӨлҸ„ кё°л°ҳ лҸ„н”Ңлҹ¬/м§Җм—° мӮ¬м „ кі„мӮ°
        predicted_delay = self._calc_propagation(ephemeris)
        predicted_doppler = self._calc_doppler(ephemeris)

        # 2. ML лӘЁлҚёлЎң мұ„л„җ кі„мҲҳ мҳҲмёЎ
        channel_coeff = self.model.forward(
            prev_csi, predicted_delay, predicted_doppler
        )
        return channel_coeff
